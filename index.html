What is Scrapbook?

Scrapbook is a collection of cache-related PHP projects. The cornerstone is key-value-store, which sets a standard interface for cache adapters to implement. All adapters hide their inner workings behind a shared implementation. Using it means you're not just building for Memcached. You code will work just fine on Redis, Couchbase or APC, without modifications! Or vice versa. Or even SQL or file cache. I'm sure you get the picture.

Having 1 shared interface to connect with all those backends means additional can be built tools on top of them & they'll work for all key-value stores. Like making them them PSR-6 compatible. Or implementing a buffered or transactional cache.

Adapters

  • Memcached
  • Redis
  • Couchbase
  • APC
  • MySQL
  • SQLite
  • PostgreSQL
  • Filesystem
  • Memory
