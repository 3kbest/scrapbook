What is Scrapbook?

Scrapbook is a caching environment for PHP. The cornerstone is key-value-store, which sets a standard interface for cache adapters to implement. All adapters hide their inner workings behind a shared implementation. Using it means you're not just building for Memcached. Your code will work just fine on any of the other adapters, without modifications!

Having 1 shared interface to connect with all those backends means additional tools can be built on top of them & they'll work for all key-value stores. Like making them PSR-6 compatible. Or implementing a buffered or transactional cache. Or stampede protection or sharding.

Adapters

  • Memcached
  • Redis
  • Couchbase
  • APC
  • MySQL
  • SQLite
  • PostgreSQL
  • Flysystem
  • Memory
